# DrCrop ICP & target list

Pilot scope: 2026 season, Northern California vineyards, under-vine weed pressure.
Goal: book 5 free audits → convert 3 to paid pilots.

## Who we target

**Tier 1 — vineyard managers / owners (decision makers).** Small-to-mid estates
(20–200 ac) where herbicide spend is visible and the manager can say yes.
Avoid mega-corp vineyards for the first wedge — procurement cycles are too long.

**Tier 2 — PCAs / crop advisors.** They already write the prescription; if one
PCA likes the tool they bring 5–20 vineyards with them. The fastest path to
volume after the first pilot.

**Tier 3 — custom applicators.** Sell them the KML/GeoJSON exports so they
can run variable-rate jobs. They pull us into client conversations.

**Tier 4 — drone pilots.** Not customers — partners. If you can't fly every
block yourself, contract local Part 107 pilots.

## Status legend

- **new** — not yet contacted
- **outreach** — first cold message sent
- **bumped** — follow-up sent
- **replied** — they responded; in conversation
- **booked** — audit confirmed on a date
- **flown** — audit complete, awaiting follow-up
- **pilot** — paid pilot signed
- **dead** — explicit no, or 3 bumps with no reply
